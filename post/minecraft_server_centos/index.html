Centos下启动Minecraft服务器

2021-02-23 Minecraft Centos

开始 —— 准备运行环境

** 安装Java环境与简单的管理软件 **

进行环境配置

  1. 安装screen，用于服务器后台管理

yum install screen

  1. 安装lrzsz，用于上传和下载

yum install lrzsz

  1. 安装Java，最基本的服务器环境

yum -y install java-1.8.0-openjdk*

开始上传

  1. 创建服务器目录

cd /usr/games
mkdir minecraft

  1. 进入/usr/games目录

cd minecraft

  1. 上传服务端文件

rz * 选择需要用到的服务端文件，如mingcraft_server_1.16.0.jar*
在弹出窗口中选择需要启动的服务端文件或整合包

开始启动服务器

  1. 开启视窗

创建并进入名为"minecraft"的视窗
screen -S minecraft

  1. 第一次启动服务端

java -Xmx1024M -Xms1024M -jar minecraft_1.14_server.jar nogui
** 启动文件名为minecraft_1.14_server.jar的服务端 *

  1. 编辑许可与vi编辑器的基本使用

第一次启动需要同意说明。

  • 在minecraft目录下操作。
  • vi eula.txt (用vi编辑器修改eula.txt文件)
  • 用键盘的方向键将光标移至文本末尾，按下A键，启动编辑模式。
  • 将“eula=false”改为“eula=true”。
  • 随后按下ESC键退出编辑模式。
  • 按住SHIFT和L键旁边的冒号键，启动命令，此时左下角会出现冒号。
  • 输入wq（w表示write，q表示quit），按下ENTER键，保存退出。
  1. 服务器参数设置

vi server.properties

  • 用vi编辑器修改服务器配置 *
    关闭正版验证可以将"online=true"改为"online=false"
    其他参数参考wiki
  1. 再次启动服务端

java -Xmx1024M -Xms1024M -jar minecraft_1.14_server.jar nogui

  • 启动文件名为minecraft_1.14_server.jar的服务端*
  • -Xmx表示服务器能用的理论最大内存量*
  • -Xms表示服务器初始时使用的内存量*
    出现Done For Help表示启动完成，在此处可输入命令
  1. 退出当前视窗

按住CTRL，随后先后分别按下A和D键，及可退出screen视窗

  1. 退出xshell

exit

退出xshell
对Minecraft后台简单管理
进入minecraft视窗

screen -r minecraft
进入名为"minecraft"的视窗
关闭服务器

进入minecraft视窗，输入stop
说明
一些注意事项

服务器重启后，视窗需要重建。
